# RenewClinic 醫美受眾分析報告

這個 repo 放置一份以客服原始對話為基礎的 HTML 簡報報告。

## 報告重點

- 聚焦醫美受眾分析：需求簇、決策心理、Persona、客戶語言與溝通策略。
- 不呈現客服流程與漏斗拆解。
- 不包含客戶姓名、電話、LINE ID、WhatsApp、微信或其他可識別個資。

## 開啟方式

直接開啟 `index.html` 即可瀏覽簡報。

鍵盤操作：

- 方向鍵 / Space：切換投影片
- `F`：全螢幕
- `O`：總覽
- `S`：講者 notes 視窗

## 檔案結構

- `index.html`：HTML 簡報主檔
- `style.css`：本報告專屬樣式
- `assets/`：html-ppt runtime、基礎樣式、字體與動畫資產
